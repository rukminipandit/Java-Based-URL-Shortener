REST API Usage

Use your API key in the X-API-Key header or ?apiKey= query param.

POST /api/v1/shorten — Create a short URL

curl -X POST https://yourhost/api/v1/shorten \
  -H "X-API-Key: lv_sk_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "originalUrl": "https://example.com/long/path",
    "customCode": "my-link",
    "analyticsEnabled": true,
    "passwordProtected": false
  }'

GET /api/v1/links — List your links

curl https://yourhost/api/v1/links?page=0&size=20 \
  -H "X-API-Key: lv_sk_your_key_here"

GET /api/v1/links/{id}/analytics — Get analytics

curl https://yourhost/api/v1/links/42/analytics?days=30 \
  -H "X-API-Key: lv_sk_your_key_here"
